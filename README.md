# EcoSentinel AI

EcoSentinel AI is an industrial sustainability monitoring system that detects anomalies in plant telemetry data and generates AI-driven sustainability recommendations.

## Features

- ML anomaly detection using Isolation Forest
- Sustainability score calculation
- Vector database using FAISS
- Retrieval-Augmented Generation (RAG)
- AI sustainability recommendations
- Automated sustainability reports

## Tech Stack

Python
Scikit-learn
Sentence Transformers
FAISS
LangChain
Google Colab

## Example Output

Plant P4 flagged for abnormal water usage.

Recommended actions:
- Inspect cooling towers
- Implement water recycling
- Check pipeline leaks
